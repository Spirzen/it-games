## Xbox Game Pass

**Xbox Game Pass** — подписочный сервис Microsoft для видеоигр. За регулярную плату подписчик получает доступ к **ротируемому каталогу**: игры можно скачивать на консоль Xbox или ПК (в зависимости от тарифа) и играть, пока тайтл остаётся в библиотеке и активна подписка. Отдельно продаётся **PC Game Pass** — тот же принцип, но каталог ориентирован на Windows. Самый полный пакет — **Xbox Game Pass Ultimate** — консольный и ПК-каталоги, онлайн-мультиплеер на Xbox, **Xbox Cloud Gaming**, **EA Play**, бонусы **Ubisoft+ Classics** и (с осени 2025) **Fortnite Crew**.

По модели сервис ближе к Netflix, чем к классической покупке диска: вы платите за доступ к библиотеке, а не за вечное владение каждой игрой. Microsoft регулярно **добавляет и убирает** сторонние релизы; эксклюзивы **Xbox Game Studios**, Bethesda и (после поглощения) Activision Blizzard в каталоге держат дольше. С января 2018 года собственные крупные релизы Microsoft изначально задумывались для Game Pass **в день розничного выхода** — с 2024–2026 годов эта привилегия закреплена за тарифом **Ultimate**, а на более дешёвых уровнях новинки студии появляются с задержкой до года.

На **27 мая 2026 года** в **Game Pass Ultimate** по сводному каталогу [newxboxone.ru](https://newxboxone.ru/game-pass-ultimate) числится **671 игра** (консоль + ПК, без учёта региональных исключений). Ниже — актуальные тарифы, хронология сервиса и полная таблица названий.

### Как устроена подписка

Подписчик **загружает полную версию** игры на устройство (консоль или ПК). Это отличает Game Pass от чистого стриминга: основной сценарий — локальный запуск с диска или SSD, без зависимости от пропускной способности канала. **Xbox Cloud Gaming** (бывший Project xCloud) даёт потоковый запуск из облака Azure на телефонах, планшетах, браузерах и Smart TV — в составе Ultimate и (с 2025 года) на уровнях Premium и Essential с ограничениями по каталогу и качеству.

Пока игра в каталоге, её можно ставить и проходить без лимита по времени. При **отмене подписки** или **выводе тайтла из библиотеки** доступ блокируется, но **сохранения** обычно сохраняются — достаточно снова оформить подписку или купить игру. На консоли допустима **офлайн-игра до 30 дней**, затем нужна проверка статуса подписки в сети.

Участники получают **скидку ~20%** на покупку игры из каталога и **~10%** на связанный DLC, пока проект остаётся в подписке. Каталог включает Xbox One, Series X|S, часть Xbox 360 и оригинального Xbox (через обратную совместимость), а на ПК — отдельный список (есть пересечения и эксклюзивы вроде *StarCraft: Remastered*).

### Тарифы на май 2026 года

Microsoft несколько раз меняла уровни и цены. С **1 октября 2025 года** действует тройка консольных планов **Essential → Premium → Ultimate** (Core и Standard переименованы). **PC Game Pass** остаётся отдельной подпиской для Windows. Цены в долларах США — ориентир Microsoft Store; в других странах суммы отличаются. **21 апреля 2026 года** компания [снизила](https://news.xbox.com/en-us/2026/04/21/xbox-game-pass-update/) стоимость Ultimate и PC Game Pass и объявила, что **новые Call of Duty** перестанут выходить в подписку в день релиза — они появятся в Game Pass **через около года**, в "следующий праздничный сезон"; уже добавленные части CoD в библиотеке остаются. По устройству экосистемы Microsoft Store см. [Microsoft Store и публикация Windows-приложений](/encyclopedia/4-code-dev/4-11-desktopnye-prilozheniya/117).

Рабочие обходные пути — подарочные коды, смена региона аккаунта, зарубежные карты или посредники; цена в рублях на вторичном рынке **не фиксирована** и сильно зависит от срока и региона кода. **Xbox Cloud Gaming** в РФ официально недоступен.

### История сервиса

| Период | Событие |
| --- | --- |
| **2013–2016** | После смены руководства Xbox (Фил Спенсер вместо Дона Маттрика) Microsoft готовит ответ на модель подписок Netflix/Spotify. Внутренний проект **Arches** превращается в Game Pass; *Sea of Thieves* закладывают как проверку экономики "день-в-день с ритейлом". |
| **28 февраля 2017** | Анонс Xbox Game Pass, закрытое тестирование в программе Xbox Insider. |
| **24 мая – 1 июня 2017** | Ранний доступ для подписчиков Xbox Live Gold, затем публичный запуск (**1 июня 2017**). Стартовый каталог — более 100 игр. |
| **Июнь 2017 (E3)** | Обратная совместимость Xbox 360/Original Xbox; часть тайтлов позже попала в Game Pass. |
| **23 января 2018** | Расширение: **релизы Microsoft Studios в день выхода** — первый тайтл *Sea of Thieves* (20 марта 2018). |
| **18 апреля – 9 июня 2019** | Анонс и запуск **Game Pass Ultimate** (Game Pass + Xbox Live Gold). Летом 2019 — **Game Pass для ПК** (бета, затем в составе Ultimate). |
| **Сентябрь 2020** | **Xbox Cloud Gaming** в Ultimate; интеграция **EA Play** (консоль и облако с ноября, ПК — с марта 2021). |
| **18 января 2022** | Более **25 млн** подписчиков; объявление покупки **Activision Blizzard** с планом ввести её игры в Game Pass после закрытия сделки. |
| **9 декабря 2021** | Ребрендинг ПК-ветки в **PC Game Pass**. |
| **2022** | League of Legends и другие проекты Riot — бонусы для подписчиков ПК/Ultimate. |
| **14 сентября 2023** | **Xbox Live Gold** сменён уровнем **Game Pass Core** (мультиплеер + небольшой каталог). |
| **12 сентября 2024** | Рост цен; уровень **Standard** ($14,99) без day-one; отказ от продажи старого "только консоль" тарифа новым клиентам; Ultimate **$19,99**. |
| **28 февраля 2023** | Расширение PC Game Pass на десятки новых стран (всего 86). |
| **1 октября 2025** | Ребрендинг: **Essential / Premium / Ultimate**; Ultimate **$29,99**; day-one только на Ultimate; рост числа игр по уровням (50 / 200 / 400+); Ubisoft+ Classics и Fortnite Crew в Ultimate; Cloud Gaming выходит из беты. |
| **21 апреля 2026** | Снижение Ultimate до **$22,99** и PC Game Pass до **$13,99**; новые **Call of Duty** — в подписке через ~год после релиза; анонс связки с Discord Nitro. |

К июлю 2025 года Microsoft заявляла о **37 млн** подписчиков Game Pass (все уровни). Источники: [Википедия](https://ru.wikipedia.org/wiki/Xbox_Game_Pass), [Xbox Wire](https://news.xbox.com/en-us/2026/04/21/xbox-game-pass-update/).

### Что входит в экосистему рядом с Game Pass

- **Xbox Game Studios / Bethesda / Activision** — основной источник day-one релизов и долгоживущих франшиз (*Halo*, *Forza*, *Starfield*, *DOOM*, *The Elder Scrolls*, *Call of Duty* в каталоге по отдельным правилам).
- **EA Play** — в Ultimate (и в PC Game Pass на компьютере) — *FC*, *Battlefield*, *Mass Effect*, *Dragon Age* и др., часто с задержкой относительно розницы.
- **Ubisoft+ Classics** — с осени 2025 в Ultimate (подмножество каталога Ubisoft).
- **Xbox Cloud Gaming** — стриминг на мобильных устройствах, браузерах, Smart TV; для Ultimate — расширенные опции качества.
- **Магазин Microsoft** — скидки подписчика на выкуп игры из каталога.

Подробнее о консоли и железе — в статье [Как устроен Xbox Series S и Series X](/encyclopedia/9-spinoff/9-03-igrovaya-industriya/11431); о подписочных моделях в индустрии — [Монетизация в играх](/encyclopedia/9-spinoff/9-03-igrovaya-industriya/115).
