## Популярные механики на Roblox

| Жанр | Петля геймплея | Типичные системы |
|------|----------------|------------------|
| **Симулятор** | Клик/сбор → апгрейд силы → новая зона | Rebirth, питомцы, автосбор |
| **Тайкун** | Строительство базы → пассивный доход | Plot claim, conveyor, unlock tiers |
| **RP** | Ролевая игра, социальные правила | Jobs, vehicles, housing |
| **Обби / паркур** | Этапы, чекпоинты, сложность | Stage, skip stage (монетизация), таймер |
| **Мини-игры** | Короткие раунды, ротация режимов | Matchmaking внутри Place |

Учебный [обби](./204) даёт базу для паркура; [королевская битва](./205) — для PvP-раундов. Симулятор и тайкун строятся на тех же **DataStore** и **серверной экономике**, но с другой петлёй прогрессии.

## Монетизация — этика и практика

Официальные каналы — **Game Pass**, **Developer Products**, **Private Servers**, **UGC** (см. [Внутриигровая экономика Roblox](./202)). Все платежи Robux проходят через API Roblox; обход ведёт к санкциям.

| Подход | Риск для опыта |
|--------|----------------|
| Косметика, ускорение прогресса без PvP | Обычно приемлемо |
| Pay-to-win в соревновательных режимах | Отток игроков, негатив отзывов |
| Loot boxes с силой в PvP | Высокий риск (уроки industry-wide) |

**Battle pass** и сезонные награды работают, если премиум-награды **не ломают** матчмейкинг для новичков.

Техническая реализация магазина в учебном проекте — [Практикум — обби на Roblox](./204); архитектура транзакций — [Внутриигровая экономика Roblox](./202).

## Маркетинг на платформе

| Канал | Что делать |
|-------|------------|
| **Качество иконки и названия** | A/B в Creator Hub, читаемость на мобильном |
| **Первые минуты (FTUE)** | Обучение без длинных текстов; первый win за 3–5 минут |
| **Retention** | События, обновления, сезоны |
| **Roblox Discovery** | Участие в программах продвижения, когда проект стабилен |
| **YouTube / TikTok** | Короткие клипы механик; коды спонсоров — по правилам платформы |
| **Сообщество** | Discord, группа Roblox, devlog в DevForum |

Метрики смотрите в **Analytics** Creator Hub.

| Метрика | Зачем |
|---------|--------|
| **D1 / D7 retention** | Возвращаются ли после первого визита |
| **Average session length** | Увлекательность петли |
| **Conversion** | Доля платящих от DAU |
| **Funnel** | Где отваливаются в FTUE (первые 5 минут) |

### Live-ops без перегруза

| Приём | Пример для обби |
|-------|-----------------|
| Еженедельный этап | `Stage` 51–60 + анонс в группе |
| Сезонный пропуск | Косметика, без pay-to-win на таймере |
| Коды в соцсетях | Разовые монеты через серверный промокод (не обход Robux API) |
| Событие x2 | Удвоение монет на выходные — флаг в `DataModule` |

Перед крупным обновлением — **закрытый playtest** Place (Friends) и проверка DataStore на альт-аккаунте.

## Чек-лист перед "релизом"

- [ ] Петля геймплея понятна за 30 секунд наблюдения
- [ ] Монетизация не ломает честный PvP (если есть)
- [ ] Иконка, описание и скриншоты обновлены
- [ ] Пройден [чек-лист раздела](./999) по Roblox
- [ ] Запланированы обновления (хотя бы 2–4 недели контента вперёд)
