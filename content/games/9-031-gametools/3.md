# Игровые магазины

<div class="article-tags">
  <span class="tag tag-notrequired">НЕ ОБЯЗАТЕЛЬНО</span>
  <span class="tag tag-beginner">ДЛЯ НОВИЧКОВ</span>
</div>

**Цифровой магазин** продаёт лицензию и доставляет файлы или ключ активации. **Лаунчер** устанавливает обновления и запускает игру. **Агрегатор** не продаёт сам — сравнивает цены у ритейлеров. **Облачный сервис** запускает игру на сервере и стримит картинку на ваше устройство.

Если нужен подробный разбор именно Steam как платформы, откройте [Steam](/encyclopedia/9-spinoff/9-03-igrovaya-industriya/11435).
По консольным витринам также есть отдельные хабы: [PlayStation Store](/encyclopedia/9-spinoff/9-03-igrovaya-industriya/11436) и [Nintendo eShop](/encyclopedia/9-spinoff/9-03-igrovaya-industriya/11437).

## PC-лаунчеры

| Магазин | DRM / запуск | Особенности |
| --- | --- | --- |
| **Steam** | Steam | Крупнейший каталог, Workshop, Proton |
| **Epic Games Store** | Epic | Бесплатные игры по расписанию |
| **GOG** | Нет (DRM-free) | Оффлайн, Galaxy |
| **EA App** | EA | Подписка EA Play |
| **Ubisoft Connect** | Ubisoft | Скидки, кросс-прогресс в части игр |
| **Battle.net** | Blizzard | WoW, Diablo, Overwatch |
| **Microsoft Store** | Microsoft | Game Pass на PC и Xbox, публикация Windows-приложений |
| **Rockstar Launcher** | Rockstar | GTA, RDR |
| **Amazon Games App** | Amazon | Prime Gaming |

---

## Инди и открытые площадки

| Площадка | Модель |
| --- | --- |
| **itch.io** | Свои цены, в т.ч. pay-what-you-want |
| **Humble Store** | Бандлы, ключи в Steam/Epic |
| **Game Jolt** | Инди, веб и загрузки |
| **IndieGala** | Ключи и DRM-free |

## Консольные магазины

| Магазин | Платформа |
| --- | --- |
| **PlayStation Store** | PS4, PS5 (+ веб для удалённой загрузки) |
| **Microsoft / Xbox Store** | Xbox и Windows |
| **Nintendo eShop** | Switch (привязка к региону консоли) |

---

## Ключи, скидки и агрегаторы

| Сервис | Тип |
| --- | --- |
| **Green Man Gaming**, **Fanatical**, **CDKeys**, **GamersGate** | Продажа ключей |
| **Plati.Market** | Маркетплейс цифровых товаров (СНГ) |
| **DLCompare**, **Allkeyshop**, **GG.deals**, **IsThereAnyDeal** | Сравнение цен |
| **KeyHub**, **Backloggd** | Учёт ключей / библиотеки (не лаунчеры) |

---

## Облачный гейминг

| Сервис | Идея |
| --- | --- |
| **Xbox Cloud Gaming** | Game Pass в браузере / приложении |
| **GeForce NOW** | Ваши игры Steam/Epic на GPU NVIDIA |
| **PlayStation Plus Premium** | Стрим PS-классики и PS4 |
| **Amazon Luna** | Подписка Amazon |
| **Boosteroid**, **Shadow** | Полный ПК или каталог в облаке |
| **Blacknut**, **GameFly**, **Antstream** | Семейный / ретро стриминг |

Требуется **стабильный интернет** и низкая задержка до дата-центра.

---

## Клиенты и веб-витрины

Многие магазины дублируются веб-сайтом: покупка в браузере, загрузка через **Steam**, **Epic Games Launcher**, **GOG Galaxy**, **EA App**, **Humble App**, **itch.io App**. Один аккаунт — библиотека на всех устройствах, где установлен клиент.
