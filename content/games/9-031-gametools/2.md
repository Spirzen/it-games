## Стрим, запись и звук

| Инструмент | Назначение | Платформы |
| --- | --- | --- |
| **OBS Studio** | Запись и трансляция, сцены, FFmpeg | Win, macOS, Linux |
| **Streamlabs Desktop** | OBS + виджеты, алерты, темы | Win, macOS |
| **NVIDIA ShadowPlay** | Instant Replay, NVENC | Windows + NVIDIA |
| **AMD ReLive** | Запись в драйвере Radeon | Win, Linux |
| **Xbox Game Bar** | Клипы, оверлей, Win+G | Windows 10/11 |
| **Bandicam**, **Fraps** | Запись / FPS (legacy) | Windows |
| **VoiceMeeter** | Виртуальный микшер для стрима | Windows |
| **Voicemod** | Изменение голоса | Windows |

## Linux: запуск Windows-игр

| Инструмент | Назначение |
| --- | --- |
| **Wine** | Слой WinAPI на Linux/macOS |
| **Proton** | Wine от Valve в Steam + DXVK/VKD3D |
| **Lutris** | Установщики Wine/Proton/эмуляторов |
| **DXVK** | DirectX 9–11 → Vulkan |
| **GameMode** | Приоритет CPU и governor для игры |
| **MangoHud** | Оверлей FPS, температур |
| **GOverlay** | GUI для MangoHud |
